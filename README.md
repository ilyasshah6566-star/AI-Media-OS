# AI Media OS

## Vision

AI Media OS is an autonomous AI-powered content creation platform.

The goal is to automate the complete content pipeline:
- Research trending topics
- Write engaging scripts
- Generate voice
- Create visuals
- Assemble videos
- Generate thumbnails
- Upload content to multiple platforms

This project is being built to learn AI engineering, automation, APIs, and software architecture while creating a real-world portfolio project.

## Tech Stack

- Python
- Git
- GitHub
- VS Code
- n8n
- SQLite
- FFmpeg
- Google AI Studio (later)
- YouTube API (later)
